# study-planner

# PC Build Helper

# Проблема
Платформа для створення та тестування конфігурацій ПК онлайн. Вона допомагає геймерам, ентузіастам або просто початківцям легко підібрати сумісні комплектуючі для ПК без необхідності глибоко занурюватися в технічні специфікації. Користувач може бути впевненим у роботі конфігурації онлайн, із-за вбудованої в платформу перевірки сумісності компонентів.

# Ціль
Створити базовий інтерфейс конфігуратора для першої ітерації розробки, який дозволяє зібрати хоча б мінімально робочу систему з "Процесору + Материнськоъ плати" з автоматичною перевіркою сумісності.

# MVP
Можливість вибору процесора та материнської плати з перевіркою їхньої сумісності. 

# Критерії приймання MVP
- Час виконання головного сценарію підбору не перевищує 1 хвилини.
- Відсутність критичних збоїв під час вибору компонентів.
- Нова людина здатна успішно завершити підбір сумісної пари без підказок та інструкцій.

# Стейкхолдери
- Користувачі - отримують максимально інтуітивно-спринятливий інтерфейс зі списками комплектуючих та автоматичною перевіркою їх сумісності.
- Розробник - оцінює успішність ітерацій, відповідає за планування та здійснення розробки.

# Ознаки успіху
1. Користувач може послідовно обрати процесор та материнську плату.
2. Авто-перевірка сумісності видає попередження у випадку, якщо користувач намагається поєднати несумісні компоненти.
3. Підсумкову робочу конфігурацію можна побачити на одному екрані.

# Користувацький сценарій
Умова - Користувач знаходиться на головній сторінці сайту
1. Натискає на кнопку "Обрати процесор" та за допомогою списку та фільтрів вибирає потрібну модель.
2. Натискає кнопку "Обрати материнську плату"
3. Бачить список плат, де відображаються лише ті, що сумісні з вже обраним процесором.
4. Обирає материнську плату
Очікуваний результат: З'являється повідомлення що збірка сумісна і користувач бачить обрану пару комплектуючих на головній сторінці.

# Початковий перелік завдань
- Зібрати базу даних нових комплектуючих винайдених з 2023-2026 роки.
- Написати логіку перевірки сумісності компонентів.
- Розробити базовий користувацький інтерфейс.
